Dark and light themes should use the same hue family for visual coherence.

Dark and light themes should share hue families, not clash with different hues. In RoleNext, the light theme uses blue-gray hues (226-228). The dark theme was originally neutral gray (hue 240). Shifting the dark theme to blue-gray (226) created visual coherence between modes. Mismatched hues between light and dark modes feel disjointed even when contrast ratios meet WCAG AA. The same hue with adjusted saturation and lightness solves both accessibility and aesthetics.
