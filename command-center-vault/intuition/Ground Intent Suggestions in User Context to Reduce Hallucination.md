Anchoring auto-suggestions to a user's resume data (extracted job titles, skills) yields higher relevance and lower hallucination than free-form suggestions.

rolenext shifted search suggestions from generic patterns to resume-driven context: the system extracts job titles from the user's resume and uses those as the suggestion source, rather than generating suggestions from scratch. This single change improved both relevance (suggestions match the user's actual background) and fidelity (no hallucinated roles the user never held). The benefit compounds: personalized suggestions drive higher adoption and better search intent quality because they reflect what the user has already done.
