Adding a single explanatory sentence below each pillar bar removes the need for users to interpret raw numbers.

The FitBreakdownBars component originally showed just numbers—e.g., '82%' under the technical pillar. Adding one-sentence explanations ('You've demonstrated Python and cloud platforms, though not the exact tool stack') made the breakdown self-explanatory. Users no longer had to guess why a pillar scored where it did. The sentence is derived from per-pillar analysis data that already existed, making it a cheap addition with high clarity payoff.
