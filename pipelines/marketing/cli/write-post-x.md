# X (Twitter)

Write a tweet (or short thread of 2-3 tweets max).

Rules:
- Punchy, direct, opinionated
- Lead with the insight, hot take, or surprising observation, not context
- If a thread: first tweet must stand alone, each tweet adds value
- Single tweet preferred if the insight fits
- No em dashes, no hashtags
- Under 280 characters per tweet
- For technical posts: tweet 1 = the problem + fix, tweet 2 = why it matters (if needed)
- For marketing posts: share the one thing that surprised you or that you'd tell another founder
