Inline category flags with actionable context beat generic template sentences for explaining why a blocker lowered the score.

WhyThisPanel's first draft showed blockers with a templated sentence: '{category} requirement but not evidenced in your background.' Every blocker had the same structure with only the category word changing. The second iteration tried category-specific explanations but they were still formulaic. The fix used inline category tags paired with context-aware tips: 'Learnable' blockers got 'Highlight any exposure on your resume to close this gap', while 'Hard requirement' blockers got a warning. The section heading 'What lowered the score' did the heavy lifting so individual items didn't need to repeat context. Result: concrete, scannable, actionable.
