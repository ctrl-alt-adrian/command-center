Four micro-interaction patterns — ghost card, instant exit, source-lane skip, custom empty state — that elevate drag-and-drop UX from functional to premium.

Kanban drag-and-drop can feel cheap if the micro-interactions aren't polished. In RoleNext's tracker, four patterns lifted the UX: (1) ghost card at 40% opacity in source lane shows where the card originated, (2) instant placeholder exit on drop prevents visual overlap with the landed card, (3) suppressing the placeholder in source lane (since the ghost already signals origin) reduces redundancy, (4) custom empty state for rejected lane ("No rejections yet 🙌") signals intent rather than failure. Each pattern solves a specific jarring moment — the cumulative effect is that drag feels intentional and feedback is clear.
