Personalizing suggestion quality assessment based on user resume context beats generic taxonomy rules.

During RoleNext search-intent quality work, personalizing the taxonomy evaluation by user background dramatically improved suggestion relevance. Rather than applying generic 'what makes a good search query' rules to everyone, the system scores quality relative to what that specific user is looking for based on their resume. This moves the quality bar from objective to contextual, catching hallucinated requirements for this person while allowing valid ones for others.
